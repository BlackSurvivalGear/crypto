# BlackStack Intelligence Terminal

**Professional Cryptocurrency Intelligence Platform**

BlackStack is a premium institutional-grade trading terminal providing real-time market intelligence, portfolio management, and whale transaction monitoring.

## Features

- **BlackStack Intelligence Bar**: Real-time market command center with institutional metrics.
- **Portfolio Manager**: Advanced analytics and performance tracking with glassmorphism UI.
- **Whale Watch**: Institutional transaction monitoring and AI-powered market insights.
- **Advanced Charting**: Integrated TradingView widgets for professional analysis.
- **Market Sentiment**: Fear & Greed index and news aggregation with sentiment analysis.

## Tech Stack

- **Frontend**: HTML5, Tailwind CSS, JavaScript (ES6 Modules)
- **Visualizations**: Chart.js, TradingView Advanced Charting
- **Data Providers**: CoinGecko, Whale Alert, Alternative.me
- **Icons**: Lucide Icons
